Github Process Flow

  • Open VS Code
  • Initialize - Create new folder structure (this stores it locally) called a repository. This is often named main.
  • Git add - stages the newly created file or folder, ready for committing,
  • Git commit -m "insert message here" - commits the stagesd file or folder to the repository.
  • Git push - this 'pushes' the newly committed changes up into the remote repository hosted on Github.
